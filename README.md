开发环境搭建
（1）下载并安装XAMPP Control Panel v3.2.4；
（2）打开Apache以使用PHP工作，打开httpd.conf文件，其中端口(port)设置为88和443；配置php.ini文件打开文件上传功能；
（3）打开MySQL以访问、使用、修改 http://localhost:88/phpmyadmin/index.php 的MySQL数据库，打开my.ini设置文件，其中端口(port)设置为3306；
（4）使用XAMPP中MySQL的Admin按钮打开MySQL数据库，创建数据库和数据表，数据库的排序规则选择“utf8_general_ci”；
（5）使用Visual Studio Code编译运行php脚本。
